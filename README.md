# Taco Tequilas Website

A professional marketing website for Taco Tequilas, built as a lightweight static site with dedicated pages for the home experience, menu, gallery, story, and visit information.

## Overview

This repository contains the front-end code for the Taco Tequilas website. The project is designed to present the restaurant brand with a clean, responsive experience and support straightforward deployment to the web.

## Vibe Coding Stack

- Visual Studio
- Codex
- Netlify
- GitHub

## Project Structure

- `index.html` - homepage
- `menu.html` - menu page
- `gallery.html` - gallery page
- `story.html` - brand story page
- `visit.html` - location and visit information
- `tacos.css` - shared site styling
- `tacos.js` - shared site interactions
- `images/` - image and media assets
